# @khoralabs/sqlite-crypto

SQLCipher helpers for Bun SQLite.

## Install

```bash
npm install @khoralabs/sqlite-crypto
```

## Usage

Import from `@khoralabs/sqlite-crypto`. See `src/` for exported APIs.

## Development

From the repository root:

```bash
bun install
bun test --cwd packages/sqlite-crypto
```

## Publishing

Use the **Publish @khoralabs/sqlite-crypto** GitHub Actions workflow. Enable **dry run** to validate the package without publishing.
